    Développement Mobile

    Qu'est ce que le Développement Mobile ?

    Le développement mobile consiste à concevoir des applications performantes pour smartphones (Android/iOS), en utilisant des technologies modernes comme Flutter. Que ce soit pour des applications professionnelles, e-commerce, éducatives ou personnelles, je crée des interfaces fluides et fonctionnelles.

    Ce que je propose :

    • Applications Android & iOS avec Flutter
    • Interfaces modernes et intuitives
    • Connexion API/Firebase & gestion de base de données
    • Débogage, optimisation et tests

  • Développement Web

    En savoir plus

    Développement Web

    Qu'est ce que le Développement Web ?

    Le développement web englobe la création de sites et applications web adaptés à tout type de besoin. Grâce à Laravel et TailwindCSS, je conçois des interfaces performantes et des fonctionnalités puissantes pour répondre à vos objectifs.

    Ce que je propose :

    • Sites vitrine et applications web dynamiques
    • Développement Front-end & Back-end
    • Intégration de bases de données MySQL
    • Sécurité, déploiement et évolutivité
